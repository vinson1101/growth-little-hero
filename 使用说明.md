# 成长小英雄 - 使用说明

## 快速开始

### 方式1：双击启动（推荐）

直接双击 `启动服务.bat` 文件，选择运行模式：

- **桌面应用模式**（默认）：在独立窗口中运行（基于Electron/Chrome）
- **网页模式**：在浏览器中打开

### 方式2：打包成独立EXE

1. 双击 `打包EXE.bat` 文件
2. 等待打包完成
3. 在 `release` 文件夹中找到 `成长小英雄 Setup 1.0.0.exe`
4. 双击安装，安装后可以在桌面或开始菜单找到快捷方式

## 开发模式

如果需要在开发模式下运行：

```bash
# 启动Vite开发服务器
npm run dev

# 启动Electron桌面应用
npm run electron:dev
```

## 打包发布

```bash
# 打包成EXE安装程序
npm run electron:build
```

打包后的文件在 `release` 文件夹中。

## 技术栈

- React 19
- TypeScript
- Vite 6
- Electron 34（桌面应用）
- Lucide React（图标）

## 注意事项

- 首次运行会自动安装依赖（需要几分钟）
- 确保已安装 Node.js
- 桌面应用模式使用 Chromium 内核（与Chrome相同）
